# User CRUD App

A simple and responsive **User Management (CRUD)** web application built with **Node.js**, **Express.js**, and **MongoDB**.  
This project is created for my learning and practice.

## 🚀 Features

- Create New Users
- Read/View All Users
- Update User Details
- Delete Users

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (with Mongoose)
- EJS Templating Engine
- Tailwind CSS
